# 🎯 Claude Project "URANGO" 생성 가이드

## 📋 Step-by-Step 가이드

### 1단계: 프로젝트 생성

1. **Claude 웹사이트 또는 앱 열기**
   - https://claude.ai

2. **좌측 사이드바에서 "Projects" 클릭**

3. **"Create Project" 버튼 클릭**

4. **프로젝트 정보 입력:**
   ```
   Name: URANGO
   
   Description: 
   아웃도어 액티비티 소셜 플랫폼 (Cycle, Bike, Run, Mountain)
   리더-팔로워 매칭 시스템, 위치기반서비스, 평가 시스템
   개발 기간: 8.5개월 | 예산: 4.3억원
   ```

5. **"Create" 클릭**

---

### 2단계: 프로젝트 지식(Knowledge) 추가

#### 옵션 A: 지식베이스 문서 업로드 (추천)

1. 프로젝트 내에서 **"Add Knowledge"** 클릭
2. **"Upload file"** 선택
3. 다음 파일 업로드:
   - `URANGO_프로젝트_지식베이스.md` ✅ (가장 중요!)
   - `URANGO_RFP.docx`
   - `URANGO_Proposal.docx`
   - `빠른업로드가이드.md`

#### 옵션 B: 직접 입력

1. **"Add Knowledge"** 클릭
2. **"Add text"** 선택
3. 아래 핵심 정보 복사하여 붙여넣기:

```markdown
# URANGO 프로젝트 핵심 정보

## 기본 정보
- 프로젝트: URANGO (유랑고) - "우리랑 Go"
- 유형: 아웃도어 액티비티 소셜 플랫폼
- 플랫폼: iOS, Android 모바일 앱
- 개발 기간: 8.5개월 (2026.3-11)
- 예산: 4.3억원

## 카테고리
1. Cycle 🚴 - 자전거
2. Bike 🏍️ - 오토바이  
3. Run 🏃 - 러닝
4. Mountain ⛰️ - 등산

## 핵심 기능
### 리더 기능
- 활동 생성 (일정/장소/난이도/인원)
- 참가자 승인/거부
- 실시간 GPS 위치 공유

### 팔로워 기능
- 검색 & 필터 (날짜/장소/난이도/인원)
- 활동 참가 신청
- 리더 팔로우

### 소셜 기능
- Follow/Follower, DM, 사진첩
- 평가 & 랭킹, LBS
- 게시판, 광고/제휴

## 기술 스택
- Frontend: React Native
- Backend: Node.js + Express
- Database: PostgreSQL + Redis
- Cloud: AWS (EC2, RDS, S3)
- Maps: Google Maps API
- Real-time: Socket.io
- Push: Firebase Cloud Messaging

## 팀 구성 (12명)
- PM 1명, 기획자 1명
- 디자이너 2명
- 백엔드 3명, 프론트엔드 3명
- QA 2명

## 개발 일정
1. 기획/분석: 3-4월 (6주)
2. 디자인: 5-6월 (8주)
3. 백엔드: 7-8월 (8주)
4. 프론트엔드: 9-10월 (6주)
5. 테스트: 10월 (2주)
6. 배포: 11월 (4주)

## 화면 구성
1. 홈: 카테고리 선택 + 추천 활동
2. 검색: 필터 + 활동 리스트
3. 생성: 활동 만들기
4. 프로필: 사용자 정보 + 통계
5. 메시지: DM

## 컬러 팔레트
- Primary: #028090 (Teal)
- Secondary: #00A896 (Seafoam)
- Accent: #02C39A (Mint)
```

---

### 3단계: 프로젝트 설정 구성

#### Instructions (선택사항)

프로젝트에 특별한 지시사항을 추가하고 싶다면:

```
이 프로젝트는 URANGO 아웃도어 소셜 플랫폼 개발을 위한 것입니다.

대화 시 다음을 기억해주세요:
- 4개 카테고리 (Cycle, Bike, Run, Mountain) 중심
- 리더-팔로워 구조
- React Native 기반 크로스플랫폼 개발
- 8.5개월 개발 일정, 4.3억원 예산
- 12명 팀 구성

문서 참조:
- 기술적 질문: 제안서 및 지식베이스 참조
- 기능 질문: RFP 및 화면 시안 참조
```

---

### 4단계: GitHub 연동 (선택사항)

Claude Projects는 GitHub와 직접 연동되지 않지만, 프로젝트 내에서 Git 관련 작업을 할 수 있습니다.

**프로젝트에 메모 추가:**
```
GitHub Repository: https://github.com/hongip/urango-project

로컬 개발 환경:
1. git clone https://github.com/hongip/urango-project.git
2. cd urango-project
3. npm install (또는 필요한 설치 명령어)
```

## 🆘 문제 해결

### 프로젝트가 안 보여요
→ 좌측 사이드바 "Projects" 섹션 확인

### 파일 업로드가 안 돼요
→ 파일 크기 확인 (제한: 10MB)
→ 지원 형식: .txt, .md, .pdf, .docx 등

### Claude가 프로젝트 정보를 참조하지 않아요
→ 프로젝트 내에서 대화하는지 확인
→ Knowledge가 제대로 추가되었는지 확인
